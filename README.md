# Vision Portfolio

Portfolio of 2D/3D vision inspection, point cloud processing, and industrial automation software projects.

> This repository summarizes selected project experience in a public and generalized form.  
> Proprietary source code, customer data, internal documents, and confidential information are not included.

## Projects

### 1. Line-scan 2D/3D Inspection System

Turntable-based inspection system using line-scan profile data and 2D image information.

**Keywords:** Line-scan, 2D/3D inspection, point cloud, image processing, Qt, C++, Python

**Summary**
- Developed software for 2D/3D data acquisition, preprocessing, inspection, and result visualization.
- Integrated camera acquisition, equipment control, and inspection logic.
- Worked on point cloud correction, image-based inspection, filtering, and visualization for production use.

---

### 2. Structured-light 3D Calibration

3D calibration and reconstruction workflow for structured-light inspection.

**Keywords:** Structured light, 3D calibration, 3D reconstruction, image processing, OpenCV, C++, MATLAB

**Summary**
- Reimplemented and optimized calibration/reconstruction workflow.
- Improved processing time and stability.
- Worked on calibration logic, image processing, and point cloud generation.

---

### 3. 3D / 2D Depalletizing Vision

Vision system combining 3D point cloud data and 2D image information for object pose estimation.

**Keywords:** 3D vision, 2D vision, robot vision, point cloud, pose estimation, coordinate transformation

**Summary**
- Processed 3D and 2D data for object localization.
- Improved pose estimation and robot picking reliability.
- Worked on preprocessing, alignment, and coordinate transformation.
